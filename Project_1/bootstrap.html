What is Bootstrap?

Bootstrap is a popular open-source CSS framework for building responsive, mobile-first websites quickly.

Key features:

  • Responsive grid system
  • Pre-styled UI components (buttons, navbars, modals, etc.)
  • Utility classes for spacing, colors, typography, and more
  • JavaScript plugins for interactive components
  • Customizable with Sass variables

Grid System

Bootstrap uses a 12-column responsive grid system with breakpoints:

  • xs (extra small: <576px)
  • sm (small: ≥576px)
  • md (medium: ≥768px)
  • lg (large: ≥992px)
  • xl (extra large: ≥1200px)
  • xxl (extra extra large: ≥1400px)
<div class="container">
  <div class="row">
    <div class="col-md-6">Half width on medium screens and up</div>
    <div class="col-md-6">Half width on medium screens and up</div>
  </div>
</div>

Components

Bootstrap provides many ready-to-use components:

Buttons

<button type="button" class="btn btn-primary">Primary Button</button>
<button type="button" class="btn btn-outline-success">Outline Success</button>

Navbar

<nav class="navbar navbar-expand-lg navbar-light bg-light">

Utility Classes

Bootstrap offers many utility classes to handle spacing, colors, display, and more.

  • m-3: margin on all sides (1rem * 3)
  • p-2: padding on all sides (0.5rem * 2)
  • text-center: center-align text
  • bg-primary: primary background color
  • d-flex: flexbox display
<div class="p-3 mb-2 bg-success text-white">Success background with padding</div>
<div class="d-flex justify-content-between">
  <div>Left</div>
  <div>Right</div>
</div>

Best Practices

✅ Use container classes for responsive fixed or fluid layouts.
✅ Use the grid system to build flexible responsive layouts.
✅ Utilize components for consistent UI.
✅ Avoid excessive overrides; use Bootstrap utility classes.
✅ Customize with Sass variables if needed.
✅ Include only needed JS plugins to reduce bundle size.
✅ Keep accessibility in mind (ARIA attributes, keyboard nav).
✅ Test responsiveness on different screen sizes.
