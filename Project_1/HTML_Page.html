About HTML

HTML stands for HyperText Markup Language. It is the standard markup language for creating web pages. It structures the content on the web (text, images, links, videos, forms, etc.). HTML is not a programming language but a markup language that uses tags.

HTML Document Structure

A basic HTML page looks like this:


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first HTML page.</p>
</body>
</html>

Breakdown:

<!DOCTYPE html>Defines document type (HTML5).
<html>Root element.
<head>Metadata (title, links, SEO, scripts).
<title>Title shown in browser tab.
<body>Visible content of the page.

HTML Elements

An HTML element has the structure:

<tagname> Content </tagname>

Types of Elements:

  1. Block-level elements → Start on a new line (e.g., <div>, <p>, <h1>–<h6>, <section>).
  2. Inline elements → Do not start on a new line (e.g., <span>, <a>, <strong>, <em>).
  3. Empty elements (void) → Don’t have closing tags (e.g., <br>, <img>, <hr>, <input>).

HTML Attributes

Attributes provide extra information about elements, written inside the opening tag:

<img src="image.jpg" alt="Description" width="300" height="200">
  • id → Unique identifier.
  • class → Used for styling (CSS/JS).
  • style → Inline CSS.
  • title → Tooltip text.
  • src, href, alt, target, etc.

HTML Head Elements

Inside <head> you typically find:

  • <title> → Page title.
  • <meta> → Metadata (charset, description, keywords).
  • <link> → External stylesheets.
  • <style> → Internal CSS.
  • <script> → JavaScript.
  • <base> → Base URL.

HTML Text Formatting

<h1> to <h6> → Headings.
<p> → Paragraph.
<br> → Line break.
<hr> → Horizontal rule.
<b>, <strong> → Bold text.
<i>, <em> → Italics.
<u> → Underline.
<mark> → Highlighted text.
<sup>, <sub> → Superscript, subscript.

HTML5 APIs

  • Canvas API → Draw graphics.
  • SVG → Scalable vector graphics.
  • Geolocation API → User’s location.
  • Drag and Drop API.
  • Local Storage & Session Storage → Store data in browser.
  • Web Workers → Run scripts in background.
