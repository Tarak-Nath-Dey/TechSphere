What is Python?

Python is a high-level, interpreted programming language known for its readability and simplicity.

Popular for:

  • Web development
  • Data science and machine learning
  • Scripting and automation
  • Scientific computing

Python Basics

Hello World

Data Types

  • int — integers, e.g., 10
  • float — decimal numbers, e.g., 3.14
  • str — text strings, e.g., "hello"
  • bool — True or False
  • list — ordered collections, e.g., [1, 2, 3]
  • dict — key-value pairs, e.g., {"name": "Alice"}

Python Syntax Essentials

  • Indentation defines code blocks (usually 4 spaces)
  • No semicolons needed to end statements
  • Comments:
    • Single-line: # This is a comment
    • Multi-line: use triple quotes """ Comment """

Control Flow

# If-else
x = 10
if x > 0:
    print("Positive")
else:
    print("Non-positive")

# For loop
for i in range(5):
    print(i)

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

Object-Oriented Programming (OOP) in Python

Classes & Objects 
class Dog:
    def __init__(self, name):
        self.name = name

    def bark(self):
        print("Woof!")

dog = Dog("Buddy")
dog.bark()
Inheritance 
class Animal:
    def speak(self):
        print("Animal sound")

class Cat(Animal):
    def speak(self):
        print("Meow")

cat = Cat()
cat.speak()
Encapsulation 
class Person:
    def __init__(self, name):
        self.__name = name  # private attribute

    def get_name(self):
        return self.__name

p = Person("Alice")
print(p.get_name())
Polymorphism 
class Bird:
    def fly(self):
        print("Flying")

class Eagle(Bird):
    def fly(self):
        print("Eagle soaring")

def let_fly(bird):
    bird.fly()

let_fly(Bird())
let_fly(Eagle())

Best Practices

✅ Follow PEP 8 style guide.
✅ Use meaningful variable and function names.
✅ Keep code DRY (Don't Repeat Yourself).
✅ Write docstrings for functions and classes.
✅ Use virtual environments to manage dependencies.
✅ Handle exceptions with try-except blocks.
✅ Write tests with unittest or pytest.
