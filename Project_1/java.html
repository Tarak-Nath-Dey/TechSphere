What is Java?

Java is a high-level, class-based, object-oriented programming language designed to have as few implementation dependencies as possible.

It is widely used for:

  • Enterprise applications
  • Android app development
  • Web servers and application servers
  • Embedded systems

Java Basics

Java programs are written in .java files and compiled into bytecode (.class files) that run on the Java Virtual Machine (JVM).

Hello World Example

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Primitive Data Types

  • int - integer numbers
  • double - floating-point numbers
  • boolean - true or false
  • char - single character

Object-Oriented Programming (OOP) Concepts

Java is fundamentally object-oriented, focusing on:

Classes & Objects

Blueprints for creating objects.

class Dog {
    String name;
    int age;

    void bark() {
        System.out.println("Woof!");
    }
}
Inheritance

Allows classes to inherit fields and methods from other classes.

class Animal {
    void eat() {
        System.out.println("Eating...");
    }
}

class Cat extends Animal {
    void meow() {
        System.out.println("Meow!");
    }
}
Encapsulation

Hiding internal state and requiring all interaction through methods.

class Person {
    private String name;

    public String getName() {
        return name;
    }
    
    public void setName(String name) {
        this.name = name;
    }
}
Polymorphism

Ability to take many forms, e.g., method overriding.

class Animal {
    void sound() {
        System.out.println("Animal sound");
    }
}

class Dog extends Animal {
    @Override
    void sound() {
        System.out.println("Bark");
    }
}

Java Syntax Essentials

  • Statements end with a semicolon ;
  • Blocks of code use curly braces { }
  • Comments:
    • Single-line: // comment
    • Multi-line: /* comment */

Control Structures

// If-else
if (x > 0) {
    System.out.println("Positive");
} else {
    System.out.println("Non-positive");
}

// For loop
for (int i = 0; i < 5; i++) {
    System.out.println(i);
}

// While loop
while (x < 10) {
    x++;
}

Methods

public int add(int a, int b) {
    return a + b;
}

Best Practices

✅ Use meaningful variable and method names.
✅ Follow Java naming conventions (camelCase, PascalCase).
✅ Keep methods small and focused.
✅ Comment your code where necessary.
✅ Handle exceptions properly using try-catch.
✅ Use access modifiers to encapsulate data.
✅ Write unit tests to verify your code.
