What is C?

C is a powerful, low-level programming language developed in the early 1970s.

Commonly used for:

  • System programming (operating systems, embedded systems)
  • High-performance applications
  • Hardware drivers
  • Game development

C Basics

Hello World

#include <stdio.h>
int main() {
    printf("Hello, World!\n");
    return 0;
}

Data Types

  • int — integer numbers
  • float — single precision floating-point
  • double — double precision floating-point
  • char — single character
  • void — no value

C Syntax Essentials

  • Statements end with a semicolon ;
  • Blocks use curly braces { }
  • Comments:
    • Single-line: // comment
    • Multi-line: /* comment */

Control Flow

// If-else
int x = 5;
if (x > 0) {
    printf("Positive\n");
} else {
    printf("Non-positive\n");
}
// For loop
for (int i = 0; i < 5; i++) {
    printf("%d\n", i);
}
// While loop
int count = 0;
while (count < 5) {
    printf("%d\n", count);
    count++;
}

Functions

int add(int a, int b) {
    return a + b;
}

Memory Management

C requires manual memory management. Key functions:

Dynamic Memory Allocation 
#include <stdlib.h>
int* ptr = (int*) malloc(sizeof(int) * 5); // allocate memory for 5 ints
if (ptr == NULL) {
    // handle allocation failure
}
free(ptr); // release memory
Pointers 
int x = 10;
int* p = &x;   // pointer to x
printf("%d\n", *p);  // dereference pointer

Best Practices

✅ Always initialize variables.
✅ Check the result of malloc() before use.
✅ Avoid memory leaks by freeing allocated memory.
✅ Use meaningful variable and function names.
✅ Keep functions small and focused.
✅ Comment your code for clarity.
✅ Use const to protect variables where appropriate.
