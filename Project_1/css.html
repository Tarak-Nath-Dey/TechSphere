About CSS

CSS stands for Cascading Style Sheets. It describes how HTML elements should be displayed (color, size, spacing, layout). CSS separates content (HTML) from presentation (design). It is used with HTML and JavaScript in the frontend web development trio.

Ways to Apply CSS

  1. Inline CSS
    2. <p style="color: red;">Hello</p>
  2. Internal CSS
    3. <style> p { color: blue; } </style>
  3. External CSS
    4. <link rel="stylesheet" href="style.css">

Selectors in CSS

Basic Selectors:
  • Universal: * { margin: 0; }
  • Element: p { color: red; }
  • Class: .box { border: 1px solid; }
  • ID: #main { background: yellow; }
  • Grouping: h1, h2 { color: blue; }
Combinators:
  • Descendant: div p { color: green; }
  • Child: div > p { color: red; }
  • Adjacent sibling: h1 + p { font-style: italic; }
  • General sibling: h1 ~ p { font-weight: bold; }
Attribute Selectors: input[type="text"] { border: 1px solid black; }
a[target="_blank"] { color: red; }
Pseudo-classes: a:hover { color: red; }
input:focus { border: 2px solid blue; }
li:first-child { font-weight: bold; }
Pseudo-elements: p::first-letter { font-size: 2em; }
p::after { content: " ✅"; }

Positioning

  • static (default)
  • relative
  • absolute
  • fixed
  • sticky

    div {
        position: absolute;
        top: 50px;
        left: 100px;
    }

Colors in CSS

  • Named colors: red, blue
  • HEX: #ff0000
  • RGB: rgb(255,0,0)
  • RGBA: rgba(255,0,0,0.5)
  • HSL: hsl(0, 100%, 50%)

Box Model

Every HTML element is a box with:

  • Content
  • Padding (space inside border)
  • Border
  • Margin (space outside border)

  div {
      width: 200px;
      padding: 10px;
      border: 2px solid black; 
      margin: 20px;
  }

Best Practices

✅ Use external CSS for maintainability. 
✅ Follow mobile-first responsive design. 
✅ Use semantic HTML + CSS for accessibility. 
✅ Avoid inline styles. 
✅ Use CSS variables for theme consistency. 
✅ Minify CSS in production for performance.

Responsive Design

  • Media Queries
    •   @media (max-width: 768px) {
    body {
      background: lightgray; 
    }
  }
  • Responsive units (%, em, rem, vh, vw)
  • Mobile-first design
